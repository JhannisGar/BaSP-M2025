# Landing Page - Biblioteca Pública José Hernández 

Este proyecto consiste en una landing page desarrollada con HTML, en un inicio para la **Biblioteca Pública José Hernández**. Su objetivo es ofrecer una presentación visual atractiva e informativa del espacio y servicios de la biblioteca, incluyendo accesos a redes sociales y medios de contacto.

-----

#  Tecnologías usadas

La página web GitHub.
La aplicación Visual Studio Code para editar el código.
Usé Git para documentar y guardar los cambios del código.

-----

#  Características del Sitio

- Diseño responsive para la versión desktop de la landing page.
- Tipografía clásica para un estilo sobrio y literario.
- Imagen principal representando el espacio físico de la biblioteca.
- Barra superior con acceso a redes sociales:
  - Email
  - Instagram
  - Facebook
- Colores suaves y cálidos (#c3e2e5 y #dc5a5a) para transmitir tranquilidad y confianza.
- Texto introductivo al objetivo de la biblioteca.

-----

#  Proceso de Desarrollo

## 1. Fundamentos HTML

Se aprendieron y aplicaron las bases de HTML  mediante los siguientes recursos:

- [HTML Crash Course For Absolute Beginners – Traversy Media (60 min)]  
- [HTML Intro – W3Schools (120 min)]  
- [HTML Cheat Sheet – Radium Rocket (10 min)]
- Además práctica básica sugerida por IA para reforzar esos conocimientos.

## 2. Control de versiones con Git y GitHub

Se utilizaron comandos básicos de Git para iniciar el repositorio, guardar versiones del proyecto y subirlo a GitHub. Recursos consultados:

- [Git & GitHub Crash Course – Traversy Media (32 min)]  
- [Explicación básica de Git & GitHub – Radium Rocket (10 min)]  
- [Using Personal Access Token with GIT and GitHub – Ed Goad (11 min)]  
- [An introduction to Git – Aditya Sridhar (45 min)]  
- [Cómo instalar Git – Ivan Medina (4 min)]
- IA

## 3. Creación del archivo README

Se aprendió cómo estructurar correctamente este archivo gracias a:

- [Creación de un archivo README – Radium Rocket (4 min)]
- Conocimientos previos sobre informes.

-----

#  Cómo visualizar la página

1. Descarga e inicia sesión en las páginas/aplicaciones anteriormente citadas.
2. Clona o descarga el repositorio.
3. Cerciorarse de tener la imagen `imagen biblio landing.jpg` en la misma carpeta que `index.html`.
4. Abre `index.html` en tu navegador.
